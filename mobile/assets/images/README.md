# BSI Logo Assets

Folder ini berisi logo Bank Syariah Indonesia yang digunakan dalam aplikasi SIMPELS Mobile.

## Files:
- `bsi_logo.png` - Logo BSI utama
- `bsi_icon.png` - Icon BSI untuk small displays

## Usage:
```dart
Image.asset('assets/images/bsi_logo.png')
```

Note: Pastikan logo BSI digunakan sesuai dengan brand guidelines BSI.
